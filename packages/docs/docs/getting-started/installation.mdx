# Installation

You can install Stan directly from your favorite package manager.

:::info
Everything is included in the base package - no extras required (e.g., installing types separately).
:::

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import CodeBlock from '@theme/CodeBlock';

<Tabs>
  <TabItem value="npm" label="npm" default>
    <CodeBlock language="sh">{`npm install --save @rkrupinski/stan`}</CodeBlock>
  </TabItem>
  <TabItem value="yarn" label="Yarn">
    <CodeBlock language="sh">{`yarn add @rkrupinski/stan`}</CodeBlock>
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    <CodeBlock language="sh">{`pnpm add @rkrupinski/stan`}</CodeBlock>
  </TabItem>
  <TabItem value="bun" label="Bun">
    <CodeBlock language="sh">{`bun add @rkrupinski/stan`}</CodeBlock>
  </TabItem>
</Tabs>

## Package exports

Some bundlers require additional configuration to handle the [exports field](https://docs.npmjs.com/cli/v11/configuring-npm/package-json#exports) in package.json:

- If you're using [Parcel](https://parceljs.org/), make sure to follow [this guide](https://parceljs.org/features/dependency-resolution/#package-exports).
