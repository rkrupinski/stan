# Examples

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

The following examples are available, grouped by tech stack:

<Tabs groupId="example-stack">
  <TabItem value="react" label="React" default>

1. [react-atom-effects](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-atom-effects) - initialize and persist `atom` values using `localStorage`.
2. [react-asynchronous-state](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-asynchronous-state) - build a deeply nested, asynchronous state graph.
3. [react-complex-state](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-complex-state) - level up with multiple dependencies.
4. [react-caching](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-caching) - apply cache policies to optimize network traffic.
5. [react-aborting-work](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-aborting-work) - abort work with Stan-provided `AbortSignal`.
6. [react-dynamic-dependencies](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-dynamic-dependencies) - watch a dynamic state graph in action.
7. [react-switching-stores](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-switching-stores) - reset all state by switching stores.
8. [react-suspense](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-suspense) - use React Suspense with Stan.
9. [react-collection-actions](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-collection-actions) - alter a dynamically referenced Stan state.

  </TabItem>
  <TabItem value="vue" label="Vue">

1. [vue-v-model](https://github.com/rkrupinski/stan/tree/master/packages/examples/vue-v-model) - bind atoms to form inputs with `v-model`.

  </TabItem>
  <TabItem value="vanilla" label="Vanilla">

1. [vanilla-state](https://github.com/rkrupinski/stan/tree/master/packages/examples/vanilla-state) - use vanilla Stan if you really have to.

  </TabItem>
</Tabs>
