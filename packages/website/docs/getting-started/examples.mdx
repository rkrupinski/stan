# Examples

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

The following examples are available, grouped by tech stack:

<Tabs groupId="example-stack">
<TabItem value="react" label="React">

- [react-atom-effects](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-atom-effects) - initialize and persist `atom` values using `localStorage`.
- [react-asynchronous-state](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-asynchronous-state) - build a deeply nested, asynchronous state graph.
- [react-complex-state](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-complex-state) - level up with multiple dependencies.
- [react-caching](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-caching) - apply cache policies to optimize network traffic.
- [react-aborting-work](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-aborting-work) - abort work with Stan-provided `AbortSignal`.
- [react-dynamic-dependencies](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-dynamic-dependencies) - watch a dynamic state graph in action.
- [react-switching-stores](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-switching-stores) - reset all state by switching stores.
- [react-suspense](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-suspense) - use React Suspense with Stan.
- [react-collection-actions](https://github.com/rkrupinski/stan/tree/master/packages/examples/react-collection-actions) - alter a dynamically referenced Stan state.

</TabItem>
<TabItem value="vue" label="Vue">

- [vue-v-model](https://github.com/rkrupinski/stan/tree/master/packages/examples/vue-v-model) - bind atoms to form inputs with `v-model`.

</TabItem>
<TabItem value="vanilla" label="Vanilla">

- [vanilla-state](https://github.com/rkrupinski/stan/tree/master/packages/examples/vanilla-state) - use vanilla Stan if you really have to.

</TabItem>
</Tabs>
