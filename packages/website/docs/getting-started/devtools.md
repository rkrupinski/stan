# Devtools

Stan comes with a dedicated Chrome extension - install it here:
https://chromewebstore.google.com/detail/stan-devtools/jioipgcofbmgbdfmdjjcmockkjkhagac

Stan DevTools let you:

- Dynamically switch between all active stores
- Inspect and track the history of state changes
- Search and filter state entries

:::info
Find out more about Stan DevTools from the [debugging guide](../guides/debugging.md).
:::
