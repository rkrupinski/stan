# Stan Skill

[Agent Skill](https://agentskills.io) for [`@rkrupinski/stan`](https://stan.party) — reusable guidance for AI coding agents working with Stan.

## Install

```sh
npx skills add rkrupinski/stan
```

Compatible with any agent that supports the [Agent Skills](https://agentskills.io) format (Claude Code, Cursor, Gemini CLI, and others).
